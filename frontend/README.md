# JDC Frontend (React + TypeScript + Vite)

This frontend uses React, TypeScript, React Router, Tailwind CSS, and React Query.

## Scripts

```bash
npm run dev
npm run build
npm run preview
npm run lint
```

## Environment Variables

The app supports both legacy `NEXT_PUBLIC_*` values and `VITE_*` values:

- `NEXT_PUBLIC_API_URL` or `VITE_API_URL`
- `NEXT_PUBLIC_WHATSAPP_NUMBER` or `VITE_WHATSAPP_NUMBER`
